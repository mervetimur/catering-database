# Catering Database

Bu proje, catering hizmetleri için geliştirilmiş bir veritabanı yapısını içerir. SQL Server üzerinde çalışacak şekilde tasarlanmış `schema.sql` dosyası ile tablolar, ilişkiler ve temel veritabanı yapısı hazırlanmıştır.

## İçerik

- `schema/` : Veritabanı tablolarının oluşturulmasını sağlayan SQL dosyaları.
- Veritabanı yapısı, catering hizmetlerine dair müşteri, sipariş ve menü bilgilerini yönetmek için optimize edilmiştir.

## Kullanım

1. SQL Server Management Studio veya benzeri bir araç ile `schema.sql` dosyasını çalıştırın.
2. Veritabanı tabloları oluşturulacak ve kullanıma hazır hale gelecektir.

## Amaç

Bu proje, catering işletmelerinin veri yönetimini kolaylaştırmak ve temel veritabanı tasarımını göstermek amacıyla hazırlanmıştır.
